GDUT 成绩提醒

作者: Chen-Dll
版本: 0.2.0

这是一个广东工业大学教务系统成绩提醒工具。它会在你的 Windows 电脑上定时、只读检查课程成绩，发现新增成绩或成绩变化后弹出系统通知。

你可以放心点击“下一步”。安装阶段只会复制程序文件、创建快捷方式和注册卸载信息，不会登录教务系统，也不会让你输入学号或密码。

安装前请先确认

1. 适用系统
   - Windows 10 / Windows 11。
   - 建议电脑已安装 Edge 或 Chrome 浏览器。

2. 安装过程不会要求账号密码
   - 安装程序只负责复制程序文件、创建快捷方式和注册卸载项。
   - 安装程序不会收集或保存学号、密码。
   - 学号、密码、验证码都不会在安装阶段输入。
   - 安装完成后，第一次打开主程序，再点击“一键配置本机”完成账号配置。
   - 如果 Windows 或杀毒软件提示“未知发布者”，这是因为当前版本还没有代码签名证书；确认来源可信后再继续安装。

3. 严格只读安全边界
   - 本工具只用于查询课程成绩。
   - 教务系统数据请求只允许访问欢迎页和成绩查询接口。
   - 导出成绩单只使用本地成绩快照生成文件，不会提交学校成绩单申请。
   - 不实现评价、修改密码、保存、删除、更新等会改变教务系统数据的功能。

4. 本地数据保存位置
   - 密码保存到 Windows 凭据管理器，不写入安装目录。
   - Cookie、配置、成绩快照和日志保存在当前用户目录下的 .gdut-grade-monitor。
   - 诊断包会隐藏学号、密码、Cookie 和完整成绩明细。

安装完成后你需要做什么

1. 勾选“启动 GDUT 成绩提醒”，或从开始菜单打开程序。
2. 在主界面点击“一键配置本机”。
3. 输入学号、密码和检查频率，默认每 30 分钟检查一次。
4. 如果弹出统一身份认证浏览器，请按页面提示完成验证码或二次验证。
5. 首次成功后只建立成绩基线，不会对已有成绩弹通知。
6. 之后有新成绩或成绩变化时，才会弹 Windows 通知。

如果第一次配置没有成功

- 先看主程序首页的提示卡片，通常会告诉你缺少浏览器、未登录或自启动未开启。
- 重新点击“一键配置本机”即可覆盖旧配置，输错密码不会写入配置文件。
- 如果弹出的统一身份认证页面要求验证码、短信或二次确认，请按学校页面提示完成。

如果安装后无法正常使用

- 打开主程序的“环境检查”页，查看缺少的环境或配置。
- 点击“导出诊断包”，发给维护者排查。
- 如果后台自启动安装失败，可以在“设置”页重新点击“安装自启动”。
