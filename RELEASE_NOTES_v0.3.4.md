# GDUT 成绩提醒 v0.3.4

这是一个后台稳定性、日志可读性和窗口关闭体验增强版。

## 更新方式

- 本次 Release 提供完整安装包和便携包；未附带 `v0.3.3 -> v0.3.4` 小补丁。
- 已安装旧版的用户可下载新版安装包覆盖安装，或下载新版便携包覆盖程序目录。
- 更新不会覆盖本地配置、密码、Cookie、成绩快照或通知密钥；这些数据仍保存在用户目录或 Windows 凭据管理器。

## 新增

- 关闭主窗口时可以勾选“不再提示，记住我的选择”。
- 设置页新增“关闭窗口时”选项，可选择：
  - 每次询问
  - 关闭按钮最小化到托盘
  - 关闭按钮退出程序
- 更新完成后第一次启动会提示“已更新到当前版本”，并说明本地数据不会被覆盖。
- 查看日志会打开面向普通用户的整理版日志，隐藏 keyring 可选后端探测噪声。

## 优化

- 设置页的“保存频率”改为“保存设置”，一次保存查询频率和关闭窗口行为。
- 后台等待下一次检查时会定期重新读取配置；把 30 分钟改成 5 分钟后，不需要等旧的 30 分钟睡眠结束。
- 手动“立即检查”完成后，会按实际完成时间重新计算下次检查时间。
- 总览页“下次检查”优先显示后台真实记录的调度时间。
- 每次检查都会写入简洁日志，说明检查时间、课程数量以及是否发现变化。
- 首次安装不会弹出“已更新”提示，避免干扰新用户的一键配置流程。

## 修复

- 修复查看日志时中文可能显示为乱码的问题。
- 修复某些环境中便携包发布冒烟测试清理临时目录过慢的问题。
- 避免未配置账号的新用户打开应用时自动触发登录错误。
