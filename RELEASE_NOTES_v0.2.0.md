# GDUT 成绩提醒 v0.2.0

这是第一个面向普通 Windows 用户发布的图形化版本。

## 下载

- `GDUTGradeMonitor-Setup.exe`：安装版，推荐使用。
- `GDUTGradeMonitor-portable.zip`：便携版，解压后运行 `GDUTGradeMonitor.exe`。

## 主要功能

- Qt/PySide6 现代桌面界面。
- 一键配置本机：保存凭据、完成登录、建立成绩基线、安装自启动。
- 默认每 30 分钟只读检查成绩，发现新增或变化成绩才弹 Windows 通知。
- 成绩分析、提醒历史、环境检查、诊断包导出。
- 本地成绩单 PDF/HTML 导出。
- 官方成绩单入口会打开学校办事大厅，由用户手动查看或下载。
- 应用内帮助页和总览新手引导。

## 安全边界

- 教务系统数据请求限制在只读 allowlist。
- 密码保存到 Windows 凭据管理器，不写入配置文件、日志或诊断包。
- 工具不会自动提交评价、成绩单申请、修改密码、保存、删除或更新教务系统数据。
