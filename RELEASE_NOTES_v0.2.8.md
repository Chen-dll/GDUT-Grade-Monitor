# GDUT 成绩提醒 v0.2.8

这是一个界面体验和稳定性修复版，重点改善总览页拥挤、运行状态显示压缩，以及登录浏览器偶发启动失败时的处理。

## 改进

- 总览页移除“新手引导”卡片，只保留“新手向导”按钮入口，避免配置完成后页面仍被说明内容占用。
- “运行状态中心”的状态卡片增加最小高度，避免被压成细线，后台状态、最近检查、下次检查等信息更容易看清。
- 首页只保留三个关键运行状态卡片，并精简“本机配置”和底部快捷按钮，减少重复信息。
- “最近变化”、首次使用向导、关于页面等内容较多的区域支持滚动显示。
- 成绩表支持双击课程打开详情，查看规范化字段和原始字段。
- 一键配置/重新登录时增加进度提示，减少长时间等待时的误解。

## 修复

- 登录浏览器启动失败时，会自动尝试主登录配置、临时登录配置、Playwright Chromium、Chrome、Edge 等候选方案。
- 如果浏览器仍无法启动，界面会显示简短中文处理建议，不再弹出大段 Playwright 底层日志。

## 验证

- `python -m unittest discover -s tests -v`
- `python -m compileall gdut_grade_monitor tests packaging docs`
- 新版 `GDUTGradeMonitor.exe` 启动冒烟测试。
