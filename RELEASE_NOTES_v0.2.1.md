# GDUT 成绩提醒 v0.2.1

这是一个登录体验补丁版本。

## 更新内容

- 登录统一认证时，程序会尽量自动勾选“7天/保持登录/免登录”选项。
- 点击“官方成绩单”时，优先使用本工具登录时的浏览器登录资料打开学校办事大厅，减少重复登录。
- 如果无法使用该登录资料打开，会自动回退到系统默认浏览器。

## 安全边界

- 官方成绩单入口仍然只负责打开学校页面。
- 工具不会自动提交成绩单申请，不调用写入接口，也不会保存办事大厅 Cookie。
- 教务成绩查询仍保持只读 allowlist。

## 已验证

- 91 个单元/集成测试通过。
- Python 编译检查通过。
- 安装包静默安装、启动、卸载 smoke test 通过。
