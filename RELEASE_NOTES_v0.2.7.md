# GDUT 成绩提醒 v0.2.7

这一版继续补产品成熟度，重点是让用户能看懂后台是否真的在工作，并能从界面或托盘快速处理常见情况。

## 新增

- 总览页新增“运行状态中心”，展示后台状态、最近检查、下次检查、登录配置、自启动状态和最近错误。
- 系统托盘新增“查看运行状态”“暂停提醒 1 小时”“恢复后台检查”“查看日志”。
- 总览页也提供暂停、恢复和查看日志快捷按钮。

## 改进

- 后台暂停时会记录心跳状态，方便判断程序仍在运行。
- 后台检查失败时会记录最近错误，成功检查后会自动清除旧错误。
- 暂停时间过期后，界面不会再误显示“暂停中”。

## 验证

- `python -m unittest discover -s tests -v`
- `python -m compileall gdut_grade_monitor tests packaging docs`
- Qt offscreen 初始化烟测：状态中心正常生成运行信息。
