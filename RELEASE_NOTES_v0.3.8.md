# GDUT 成绩提醒 v0.3.8

这是一个成绩页体验优化版，主要解决成绩页上方两张图表占用空间后，表格一次只能看到少量课程的问题。

## 更新方式

- 已安装 `v0.3.7` 的打包版用户，可以在“设置 -> 检查更新”里安装 `v0.3.8` 小补丁。
- 小补丁只替换应用文件，不会覆盖本地配置、密码、Cookie、成绩快照、日志或通知密钥。
- 如果不使用小补丁，也可以下载新版安装包或便携包覆盖使用。

## 新增

- 成绩页新增 `收起图表` / `展开图表` 按钮。
- 收起图表后，成绩表会获得更多显示空间，一次可以看到更多课程。
- 展开和收起使用平滑动画，避免页面突然跳动。

## 说明

- 这个版本只调整本地界面显示，不改变教务系统查询逻辑。
- 严格只读边界保持不变，不会新增任何会修改教务系统数据的请求。
