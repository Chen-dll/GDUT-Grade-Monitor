# GDUT 成绩提醒 v0.3.7

这是一个成绩页体验增强版，加入“本地手动补录成绩”能力，适合还没出成绩、显示 0 分、或需要自己临时估算绩点的场景。

## 更新方式

- 已安装 `v0.3.6` 的打包版用户，可以在“设置 -> 检查更新”里安装 `v0.3.7` 小补丁。
- 小补丁只替换应用文件，不会覆盖本地配置、密码、Cookie、成绩快照、日志或通知密钥。
- 如果不使用小补丁，也可以下载新版安装包或便携包覆盖使用。

## 新增

- 成绩页新增 `手动补录`：选中 0 分或未出成绩课程后，可以填写本地估算成绩。
- 成绩页新增 `撤销补录`：选中已补录课程后，可以删除本地补录并恢复官方原始显示。
- 手动补录成绩会在表格中以浅橙色标记，并显示来源为“手动”。
- 成绩详情和本地成绩单导出会标注“手动补录，非官方”。
- 后台检查发现教务系统登录状态失效时，会发送通知提醒你打开应用重新登录。

## 规则

- 手动补录只保存在本机，不会修改教务系统，不会上传。
- 如果教务系统之后检测到官方成绩大于 0，会自动以官方成绩为准。
- 官方等级成绩例如“优秀”不会被手动补录覆盖。
- 疑似缓考旧记录的 0 分行不会被手动补录替换，也不会计入总学分。
- 重新登录提醒会做节流处理，同一次登录失效不会一直重复轰炸通知。
